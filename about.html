About

เว็บไซต์นี้จัดทำขึ้นเพื่อเป็นส่วนหนึ่งของวิชา Web-based Geographic Information System เพื่อให้ได้ศึกษาและนำมาปฏิบัติเกี่ยวกับความรู้ในเรื่องของ Web Map Application โดยได้ศึกษาหาความรู้ผ่านการเรียนการสอนในชั้นเรียนและค้นคว้าผ่านแหล่งข้อมูลต่าง ๆ เช่น หนังสือ อินเทอร์เน็ต บทความวิชาการ เป็นต้น โดยในรายงานเล่มนี้มีเนื้อหาเกี่ยวกับหลักการวิเคราะห์ ออกแบบ และพัฒนา ระบบภูมิสารสนเทศเพื่อนำมาประยุกต์บนเว็บไซต์ คณะผู้จัดทำได้เลือกหัวข้อนี้ในการทำรายงาน เนื่องมาจากจำนวนผู้ติดเชื้อโควิด-19 ที่เพิ่มมากขึ้นในการระบาดระลอกที่ 3 ในประเทศไทย ทำให้ต้องมีการเพิ่มมาตรการรองรับการดูแลผู้ป่วยในหลากหลายแนวทาง และเพื่อให้ง่ายต่อการค้นหาโรงพยาบาลและโรงพยาบาลสนามในพื้นที่ ผู้จัดจึงได้ทำการพัฒนา Web Map Application ผ่านการเรียนรู้ในรายวิชาที่ศึกษา และคณะผู้จัดทำคาดหวังเป็นอย่างยิ่งว่าการจัดทำรายงานฉบับนี้จะมีข้อมูลที่เป็นประโยชน์ต่อบุคคลที่สนใจจะศึกษาเกี่ยวกับ Web Map Application เป็นอย่างดี
